# SmartMeal

SmartMeal – Planificador Inteligente de Comidas

## Descripción

SmartMeal es una aplicación diseñada para ayudar a los usuarios a planificar sus comidas de manera sencilla e inteligente.
La aplicación permite organizar menús, visualizar ideas de comidas y facilitar la planificación semanal de alimentos.

El objetivo principal es ayudar a las personas que no saben qué cocinar o cómo organizar sus comidas durante la semana.

## Funcionalidades

* Visualización de comidas o recetas
* Organización de menús
* Interfaz sencilla para explorar opciones de comida
* Planificación básica de alimentos
